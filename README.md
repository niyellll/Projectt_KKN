# Clear Light Stories

A modern, accessible storytelling website template for people with low vision.  
The project is fully static and can be uploaded directly to GitHub or deployed with GitHub Pages.

## Features

- Clean white background with high-contrast dark text
- Dark blue headings and navigation
- Purple buttons and interactive components
- Yellow accent used sparingly for emphasis and focus states
- Sticky navigation with large clickable menu items
- Large readable sans-serif typography
- Comfortable line spacing and paragraph spacing
- Hero section with featured image and story title
- Alternating story sections with large images and descriptive captions
- Semantic HTML structure
- Skip link for keyboard users
- Visible focus indicators
- Adjustable text size controls
- High contrast mode toggle
- Responsive layout for desktop, tablet, and mobile
- No external dependencies

## How to Use

1. Upload all files to a GitHub repository.
2. Open `index.html` in a browser to preview the website.
3. Replace the sample text in `index.html` with your own story.
4. Replace the SVG files inside `assets/` with your own photos or illustrations.
5. Keep descriptive `alt` text and captions for every image.

## Deploy to GitHub Pages

1. Go to your repository on GitHub.
2. Open **Settings**.
3. Open **Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Choose your main branch and root folder.
6. Save and wait for GitHub Pages to publish the site.

## Accessibility Notes

This template is designed around WCAG-inspired accessibility practices for low-vision users:

- Strong text contrast
- Large default font size
- Clear heading hierarchy
- Keyboard-friendly navigation
- Large buttons and navigation targets
- Visible focus indicators
- Descriptive image alternatives
- Captions for visual content
- Consistent spacing and alignment
- Reduced visual clutter
- Important information is not communicated by color alone

Before final publication, test the website with real users, browser zoom, keyboard navigation, and accessibility checkers.
